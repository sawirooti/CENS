## Объявления include/Definition.hpp

Тут все параметры: размеры комнаты, шаг дискретизации карты, пины, скорость машинки и пр.
Всякие подобные штуки тоже закидывай, если будут.

## Функции для управления:

- *Movement::Forward();* - едет вперед
- *Movement::Back();* - едет назад
- *Movement::TurnRight();* - поворачивает направо (танковый способ)
- *Movement::TurnLeft();* - поворачивает налево (танковый способ)
- *Movement::Stop();* - жестко останавливается (колеса нельзя прокрутить)
- *Movement::Neutral();* - нейтраль, колеса могут прокручиваться

Скорость колёс фиксированная, я не очень хочу с ней баловаться, но если нужно будет, то добавлю сеттер.

## Взаимодействие с КЭНС

 Вызываешь: *CENS::calculatePosition(float distanceFront, float distanceRight)*. Внутрь закидываешь дальности до передней и правой стены соответственно.
 Она тебе вернёт следующее:

```C++
 struct CENSResult
{
    float x;
    float y;
    bool valid;
};
```

По полю valid оцениваем, справился ли КЭНС вообще со своей задачей или там внутри что-то сломалось
